# Jeu du Simon
Page Web permetant de jouer au jeu du Simon.

## Description
Jeu du Simon où il faut répéter la séquence dans le bon ordre afin de marquer des points.

L'objectif de ce projet est d'apprendre à utiliser les différentes technologies Web afin de créer un jeu simple.

*Projet universitaire*

## Fonctionnalités
- Son des 4 boutons
- Système de comptage du score (suite la plus grande)
- Temps que le joueur prend à répéter la séquence

## Technologies
- [`HTML`][1] - Structure de la page
- [`CSS`][2] - Style de la page
- [`JavaScript`][2] - Logique de la page

git clone https://github.com/MrYXeL/git_projet_web
